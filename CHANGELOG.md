# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-10-18

### Added
- Initial release of pyswiftdemangler
- Core `demangler` class for demangling Swift symbol names
- Command-line interface for symbol demangling
- Test mode for verifying demangling functionality
- Windows support via ctypes and Swift DLL
- PyPI packaging configuration
